## 美团买菜版本

这个脚本适用于美团买菜 v5.32.0，不要用最新的版本。
旧版本的获取方法：
下载一个”豌豆荚“app，或者在网页上搜索豌豆荚，然后在里面搜索应用名称，最下面会有一个“历史版本”，找到 v5.32.0 并安装，注意安装前需要先卸载掉美团买菜。
